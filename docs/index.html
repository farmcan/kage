Desktop Workspace

Browse project sessions, search recent prompts, launch continue, fork, bridge, or replay story actions, and explore sanitized demo sessions before you have local history.

Rescue Context

Locate the coding session where the important decision, bug, or plan actually happened.

Bridge Agents

Move a useful session from Claude Code to Codex, Codex to QoderCLI, or any supported route.

Replay Work

Turn a transcript into a local story view so a session can be reviewed instead of guessed.

Use KAGE if

  • You use more than one AI coding agent.
  • You need to find old local sessions by project or transcript text.
  • You want to fork, bridge, or replay sessions without copy-pasting transcripts.

Skip it for now if

  • You only use one agent and never revisit old sessions.
  • You need a signed and notarized macOS installer today.
  • You expect hosted sync, team sharing, or mobile control today.

One local workspace for agent memory

Each agent has its own native session store. KAGE keeps the local-first model, then adds a desktop workspace, discovery, conversion, preview, cleanup, and replay on top.

Route Use it when
kage c2x Continue a Claude Code session in Codex.
kage x2q Move Codex context into QoderCLI resume format.
kage c List matching Claude sessions for the current project.
kage q2v Replay a QoderCLI session as a local HTML story.

Not another agent or terminal

Codex, Claude Code, QoderCLI, Warp, and Terminal still run the work. KAGE manages the local session layer around them: discovery, search, continue, fork, bridge, replay, and cleanup.

Native resume Best for one agent's own sessions. KAGE adds cross-agent search and bridge workflows.
Warp / Terminal Best for running shells. KAGE keeps the memory map of what happened across agents.
Cloud tools Best for hosted team search. KAGE keeps transcript content local by default.

Product direction

KAGE is becoming a practical control surface for local AI coding work.

Desktop

Browse sessions, filter agents, inspect context, run per-session actions, and use demo mode for first-run exploration.

Search

Search local sessions by project, agent, title, and transcript text.

Bridge

Move useful context across Claude Code, Codex, and QoderCLI native sessions.

Local First

Keep transcripts on disk and use the CLI JSON contract as the app boundary.
