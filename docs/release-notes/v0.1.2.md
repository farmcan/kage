# KAGE v0.1.2

KAGE v0.1.2 clarifies the desktop action model around session branching.

## What Changed

- Added a first-class `Fork as new session` action for same-agent branching.
- Renamed `Replay` to `Open replay story` in the desktop app and action contract.
- Opening a replay story from the desktop or menu bar now launches the generated local HTML file.
- Result banners now distinguish forked sessions, bridged sessions, resume commands, and replay stories.
- README now documents the difference between `Continue`, `Fork`, `Bridge`, and `Replay story`.

## Install

Download the unsigned macOS DMG:

```text
https://github.com/farmcan/kage/releases/download/v0.1.2/KAGE-0.1.2.dmg
```

The DMG is unsigned for now. If macOS blocks the first launch, right-click `KAGE.app`, choose `Open`, then confirm.
