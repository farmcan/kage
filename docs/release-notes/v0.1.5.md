# KAGE v0.1.5

KAGE v0.1.5 is a first-run and adoption polish release.

## What's New

- Added a launch-ready demo flow asset for the README and GitHub Pages homepage.
- Improved the desktop empty state for users who have not created matching local agent sessions yet.
- The empty state now offers refresh, include-subdirectories, directory selection, and getting-started actions.
- Added public roadmap issues for the real demo GIF and sanitized demo sessions.

## Download

Download the unsigned macOS DMG:

```text
https://github.com/farmcan/kage/releases/download/v0.1.5/KAGE-0.1.5.dmg
```

The DMG is unsigned for now. If macOS blocks the first launch, right-click `KAGE.app`, choose `Open`, then confirm.

Signing and notarization are tracked separately in GitHub issue #34 because they require Apple Developer ID credentials.
