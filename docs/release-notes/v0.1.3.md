# KAGE v0.1.3

KAGE v0.1.3 is a patch release for Claude Code session discovery.

## What Changed

- Fixed issue #31 by skipping Claude `subagents/` transcript directories during session discovery.
- Prevented KAGE from generating `claude --resume <id>` actions for sub-agent transcript folders that Claude Code cannot resume as top-level conversations.
- Added a regression test that creates a newer sub-agent transcript next to normal Claude sessions and verifies actions only target resumable sessions.

## Install

Download the unsigned macOS DMG:

```text
https://github.com/farmcan/kage/releases/download/v0.1.3/KAGE-0.1.3.dmg
```

The DMG is unsigned for now. If macOS blocks the first launch, right-click `KAGE.app`, choose `Open`, then confirm.
