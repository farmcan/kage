# KAGE v0.1.6

KAGE v0.1.6 is a first-run demo mode release for the macOS desktop app.

## Highlights

- Added opt-in sanitized demo sessions for Codex, Claude Code, and QoderCLI.
- The no-session empty state now offers `Explore Demo` so new users can understand the workspace before they have local agent history.
- Demo mode keeps actions safe: continue, fork, bridge, and replay are previews and do not read or write private transcript files.
- Demo sessions still exercise project grouping, agent filters, transcript search, and the embedded terminal preview surface.
- Replaced the system empty-state action layout with a custom desktop empty state so action buttons do not truncate.
