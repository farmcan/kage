# KAGE v0.1.17

KAGE v0.1.17 aligns the product UI and packaging surface after the latest polish pass:

- Added a production-ready "search → bridge" demo GIF to the top of README and homepage.
- Refreshed homepage/demo assets and documentation for the current multi-agent web/desktop workflow.
- Consolidated release metadata to point to the v0.1.17 DMG and updated CLI version strings.
- Kept CLI behavior unchanged from v0.1.16 while continuing to enforce local safety checks for transcript access and session actions.

## Release Notes

- `serve` now keeps mobile interactions snappier under larger workloads and uses the improved dispatch/board state patterns from v0.1.16.
- Homepage and app docs now reflect the running release line and the new task-board-first mobile behavior.
- Existing unsigned-app guidance remains: enable `KAGE_CODESIGN_IDENTITY` and `KAGE_NOTARY_PROFILE` in CI/release workflows when generating signed/notarized distributables.
