# KAGE v0.1.14

KAGE v0.1.14 fixes the desktop update check discovered during local reinstall testing.

## Desktop

- Keeps the GitHub Releases API as the primary update source.
- Falls back to the GitHub Releases `latest` redirect when unauthenticated API requests are rate-limited.
- Preserves the Settings update status flow: checking, up to date, update available, or error.
