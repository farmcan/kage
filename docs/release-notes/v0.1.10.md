# KAGE v0.1.10

KAGE v0.1.10 keeps the desktop app responsive when local session history grows.

## What's Changed

- Loaded a recent 90-day / 120-session window by default in the desktop workspace.
- Added an explicit full-history mode with older sessions folded by month.
- Added New Session launch controls for Codex, Claude Code, and QoderCLI.
- Added a Terminal.app launch path for noisy interactive sessions that feel better outside the embedded terminal.
- Added `--limit` support for `kage sessions`, `kage search`, and `kage actions`.
- Kept desktop search aligned with the same recent-history scope used by the sidebar.
- Updated README, docs homepage, and app install links for the v0.1.10 DMG.
