# KAGE v0.1.4

KAGE v0.1.4 is a desktop polish release for the public macOS DMG.

## What's New

- Added a real macOS `.icns` app icon to the bundle and set `CFBundleIconFile`.
- Added a KAGE template icon for the normal menu bar state.
- Replay story actions now open the generated HTML directly and still offer Show in Finder.
- Search empty states now explain the active agent, query, and directory filters, with quick recovery actions.

## Download

Download the unsigned macOS DMG:

```text
https://github.com/farmcan/kage/releases/download/v0.1.4/KAGE-0.1.4.dmg
```

The DMG is unsigned for now. If macOS blocks the first launch, right-click `KAGE.app`, choose `Open`, then confirm.

Signing and notarization are tracked separately in GitHub issue #34 because they require Apple Developer ID credentials.
